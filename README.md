# cricket-auction
This is an application for online cricket auction system


# UI
1. yarn install & yarn start
2. http://localhost:3000/login
3. username: admin
4. password: cricket123


# Backend
1. Open a terminal and go to the backend folder:
      **cd C:\cricket-auction-backend**
2. (Recommended) Create/activate a virtual env:
   **python -m venv venv
     venv\Scripts\activate**
3. **pip install -r requirements.txt**
4. **python app.py**
5. It binds to http://127.0.0.1:5001 by default (see app.run(port=5001)).
